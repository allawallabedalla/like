Abseits. Unterkünfte, die ihren eigenen Weg gehen — Google + OpenStreetMap.

Gib oben einen Ort ein. Das Tool durchsucht den Kartenbereich nach Pensionen, Gästehäusern, Ferienwohnungen & Co.

Quellen

Anzeigen

Achtung: blendet alle Einträge ohne Website/Telefon aus — betrifft v. a. Camping-, Wildnis- und Schutzhütten.

Unterkunftstyp

Google-API-Key

Lokal ist der Key sicher. Einrichtung in der Google Cloud Console: Projekt anlegen → Maps JavaScript API und Places API (New) aktivieren → API-Key erstellen → Abrechnung aktivieren. Tipp: Key auf Referrer http://localhost:8000/* beschränken und Budget-Limit setzen. Ohne Key läuft nur OSM.

App installieren

Am iPhone in Safari öffnen → Teilen-Symbol → „Zum Home-Bildschirm". Danach startet „Abseits" als eigenes App-Icon im Vollbild — ohne Browser-Leiste. Funktioniert, sobald die Seite über https (z. B. GitHub Pages) oder localhost läuft.
